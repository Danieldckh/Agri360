Interaction Flow

1. Hover icon → icon highlights, entire dashboard content fades (opacity ~0.2)
2. Radial items stagger into view in an arc from the icon's position
3. Faint lines connect items back to the origin point
4. Hover a radial item → it highlights, click to execute the action
5. Move mouse away → radial closes, dashboard un-fades
